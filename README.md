# 📚 Wiki-GitHub

Aplicação em **ReactJS** para explorar e organizar repositórios do GitHub com uma interface moderna e responsiva.  
Projeto desenvolvido como parte do curso da **Plataforma DIO.me**.

---

## 🚀 Funcionalidades
- 📖 Modal interativo com seções **Sobre** e **Contato**  
- 👨‍💻 Navegação adaptada para **desktop** e **mobile** (menu hambúrguer estilizado)  
- 🎨 Interface construída com **Styled Components**  
- 🌐 Consistência visual: azul claro como padrão + hover elegante  
- ⚡ Performance otimizada e experiência fluida  

---

## 🛠️ Tecnologias utilizadas
- [React](https://reactjs.org/)  
- [Styled Components](https://styled-components.com/)  
- [GitHub API](https://docs.github.com/en/rest)

---

## 🎯 Objetivo
Este projeto foi criado como prática em React, mas também como uma ferramenta útil para quem trabalha diariamente com o GitHub e deseja explorar repositórios de forma simples e organizada.
