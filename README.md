# Wiki-GitHub
Aplicação em React para explorar repositórios do GitHub. Interface moderna com Styled Components, modal interativo e navegação responsiva (desktop e mobile).

🚀 Wiki GitHub
Aplicação desenvolvida em React para explorar e organizar repositórios do GitHub com uma interface moderna e responsiva.

✨ Recursos
- 📖 Modal interativo com seções Sobre e Contato
- 👨‍💻 Navegação adaptada para desktop e mobile (menu hambúrguer estilizado)
- 🎨 Interface construída com Styled Components
- 🌐 Consistência visual: azul claro como padrão + hover elegante
- ⚡ Performance otimizada e experiência fluida
🛠️ Tecnologias
- React
- Styled Components
- GitHub API

🎯 Objetivo
Projeto criado como prática em React, mas também como uma ferramenta útil para quem trabalha diariamente com GitHub.
